# 简单发布相关命令

# new
> $ hexo new [title]

新建一篇文章。如果标题包含空格的话，请使用引号括起来。

# generate
> $ hexo generate

生成静态文件。
> -d, --deploy	文件生成后立即部署网站

> -w, --watch	监视文件变动

# server
> $ hexo server

启动服务器。默认情况下，访问网址为： http://localhost:4000/。
